# Vinuvisthara Backend

Laravel 11 API backend for Vinuvisthara application.

## Deployment

This repository is deployed on Railway using native PHP build.

### Environment Variables
- APP_ENV=production
- APP_DEBUG=false
- APP_URL=https://api.vinuvisthara.com
- DB_CONNECTION=postgresql
- (Database credentials provided by Railway)

### Features
- RESTful API
- Filament Admin Panel
- PDF Generation
- Payment Integration (Razorpay)
